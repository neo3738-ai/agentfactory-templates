# Refactoring Advisor

Analyze code and suggest refactoring improvements

## Pipeline

Detect Smells -> Suggest Refactoring

## Agents

1. **Code Smell Detector** (analyst): Identify code smells and anti-patterns
2. **Refactoring Advisor** (advisor): Suggest specific refactoring strategies

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
