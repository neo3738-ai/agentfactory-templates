# Learning Roadmap Builder

Build a structured learning roadmap for any skill

## Pipeline

Research Path -> Build Roadmap

## Agents

1. **Skill Mapper** (researcher): Research the skill landscape and best learning paths
2. **Roadmap Builder** (planner): Build a structured learning roadmap with milestones

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
