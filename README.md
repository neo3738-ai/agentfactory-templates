# Agentfactory Templates

Community-maintained template library for [Agentfactory](https://github.com/neo3738-ai/agentfactory) — the AI agent pipeline builder.

## What Are Templates?

Templates are pre-built agent pipelines, individual agents, and councils you can install with one click from Agentfactory's Marketplace tab. Each template defines agents, their roles, tools, and how they work together to accomplish a task.

## Quick Start

1. Open Agentfactory
2. Go to the **Marketplace** tab
3. Browse or search for a template
4. Click **Install**
5. The template appears in your Home tab — ready to use

## Share Your Own

**From the app (v0.4.6+):** Click the Share button on any pipeline, agent, or council. A pull request is created automatically.

**Manual:** Fork this repo, add your template to `community/<category>/<id>/`, and open a PR. See [CONTRIBUTING.md](CONTRIBUTING.md).

## Import a Template Manually

1. Download the `template.json` file from any template folder
2. In Agentfactory, go to Marketplace > My Templates
3. Click **Import from File**
4. Select the downloaded JSON file
