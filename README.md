# Agentfactory Templates

Community-maintained template library for [Agentfactory](https://github.com/neo3738-ai) — the AI agent pipeline builder.

## What Are Templates?

Templates are pre-built agent pipelines you can install with one click from Agentfactory's Marketplace tab. Each template defines a set of agents, their roles, tools, and how they work together to accomplish a task.

## Quick Start

1. Open Agentfactory
2. Go to the **Marketplace** tab
3. Browse or search for a template
4. Click **Install**
5. The template appears in your Home tab — ready to use

## Import a Template Manually

1. Download the `template.json` file from any template folder
2. In Agentfactory, go to Marketplace > My Templates
3. Click **Import from File**
4. Select the downloaded JSON file
