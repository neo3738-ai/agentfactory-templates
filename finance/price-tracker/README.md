# Price Tracker

Track prices on products and get notified of drops

## Pipeline

Check Prices -> Analyze Deals

## Agents

1. **Price Checker** (fetcher): Check current prices on specified products
2. **Deal Analyzer** (analyst): Analyze price history and determine if it's a good deal

**Schedule:** `0 10 * * *`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
