# Hobby Finder

Discover new hobbies based on your interests and lifestyle

## Pipeline

Research Hobbies -> Recommend

## Agents

1. **Hobby Researcher** (researcher): Research hobbies matching the user's profile
2. **Hobby Advisor** (advisor): Recommend hobbies with getting-started guides

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
