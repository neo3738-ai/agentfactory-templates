# Gift Idea Finder

Find personalized gift ideas based on the recipient's interests

## Pipeline

Research Gifts -> Curate List

## Agents

1. **Gift Researcher** (researcher): Find gift ideas based on recipient's interests and budget
2. **Gift Curator** (writer): Curate and present the best gift options

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
