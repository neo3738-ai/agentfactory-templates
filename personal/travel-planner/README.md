# Travel Itinerary Planner

Plan detailed travel itineraries with activities and logistics

## Pipeline

Research -> Build Itinerary

## Agents

1. **Destination Researcher** (researcher): Research the destination for activities, food, and logistics
2. **Itinerary Builder** (planner): Build a detailed day-by-day itinerary

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
