# Weekly Meal Planner

Plan meals for the week with recipes and grocery lists

## Pipeline

Plan Meals -> Recipes & List

## Agents

1. **Meal Planner** (planner): Create a weekly meal plan based on preferences
2. **Recipe & List Writer** (writer): Write recipes and compile a grocery list

**Schedule:** `0 10 * * 0`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
