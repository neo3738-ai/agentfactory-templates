# Email Sequence Planner

Plan multi-email sequences for onboarding, sales, or nurture campaigns

## Pipeline

Plan Strategy -> Write Emails

## Agents

1. **Sequence Strategist** (strategist): Plan the email sequence strategy
2. **Sequence Writer** (writer): Write all emails in the sequence

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
