# Daily Tech News Briefing

Get a summary of top tech news delivered to you

## Pipeline

Research -> Summarize

## Agents

1. **News Researcher** (researcher): Find top tech news stories from configured sources
2. **News Summarizer** (writer): Summarize news stories into a concise briefing

**Schedule:** `0 9 * * *`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
