# Weather & Commute Briefing

Get a morning briefing with weather forecast and commute conditions

## Pipeline

Get Weather -> Write Briefing

## Agents

1. **Weather Agent** (researcher): Fetch current weather and forecast for user's location
2. **Briefing Writer** (writer): Create a concise morning briefing

**Schedule:** `0 6 * * *`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
