# Event & Conference Tracker

Track upcoming events, conferences, and meetups in your field

## Pipeline

Scan Events -> Curate

## Agents

1. **Event Scanner** (researcher): Scan for upcoming events and conferences
2. **Event Curator** (writer): Curate and organize event listings

**Schedule:** `0 9 * * 1`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
