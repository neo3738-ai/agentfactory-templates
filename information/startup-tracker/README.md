# Startup Funding Tracker

Track recent startup funding rounds and acquisitions

## Pipeline

## Agents

1. **Funding Scanner** (researcher): Scan for recent funding announcements
2. **Funding Reporter** (writer): Write a funding activity report

**Schedule:** `0 9 * * 1`
