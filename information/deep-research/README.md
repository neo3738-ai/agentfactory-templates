# Deep Research Report

Research a topic in depth and produce a detailed report

## Pipeline

Research -> Analyze -> Write Report

## Agents

1. **Researcher** (researcher): Gather comprehensive information on the given topic
2. **Analyst** (analyst): Analyze gathered research and identify key insights
3. **Report Writer** (writer): Write a structured, detailed report from the analysis

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
