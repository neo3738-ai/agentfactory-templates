# ArXiv Paper Digest

Get daily summaries of new papers in your research areas

## Pipeline

Fetch Papers -> Summarize

## Agents

1. **Paper Fetcher** (researcher): Fetch recent papers from ArXiv in specified categories
2. **Paper Summarizer** (writer): Create readable summaries of academic papers

**Schedule:** `0 8 * * *`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
