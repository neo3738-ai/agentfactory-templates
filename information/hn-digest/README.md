# Hacker News Daily Digest

Get a daily digest of top Hacker News stories and discussions

## Pipeline

Fetch HN -> Digest

## Agents

1. **HN Fetcher** (researcher): Fetch top stories from Hacker News
2. **HN Summarizer** (writer): Create a readable HN digest

**Schedule:** `0 18 * * *`

## Usage

Use this template from the Agentfactory Home tab or install it from the Marketplace.
